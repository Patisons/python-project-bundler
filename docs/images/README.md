# Screenshots

This folder contains screenshots for the documentation.

## Required Screenshots

1. **main_window.png** - Main application window
2. **file_selection.png** - File selection dialog during restoration

## How to Add Screenshots

1. Run `project_bundler.py`
2. Take screenshots of the windows mentioned above
3. Save them with exact names listed above
4. Place them in this folder
5. Commit and push to GitHub:
   ```bash
   git add docs/images/*.png
   git commit -m "Add application screenshots"
   git push
   ```

## Screenshot Guidelines

- **Format**: PNG (recommended) or JPG
- **Size**: Reasonable resolution (not too large)
- **Content**: Show clean, representative examples
- **Language**: English UI preferred (if possible)

---

**Note:** Screenshots are optional but recommended for better documentation.
